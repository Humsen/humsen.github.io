windows10 配置 Java+Maven环境变量

2020-12-12 /
  • Windows
 / 点击 / 阅读耗时 1 分钟

1、Java 环境变量

不使用%JAVA_HOME%，防止出错

测试
在cmd下输入 java -version java javac，都有输出则成功，特别是javac，一定要有输出！

2、Maven 环境变量

M2_HOME和MAVEN_HOME两个都需要，有的项目会用到

测试
在cmd下输入mvn -v，有输出即可
