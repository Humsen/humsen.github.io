windows 10 隐藏本地磁盘

2020-12-11 /
  • Windows
 / 点击 / 阅读耗时 1 分钟

1、打开本地组策略编辑器

crtl+R -> gpedit.msc -> 用户配置 -> 管理模板 -> Windows组件 -> 文件资源管理器

2、在文件资源管理器中找到 隐藏‘我的电脑’中的这些指定的驱动器，双击打开。
先点击选择“已启用”，然后点击选择下列组合的一个,选择“仅限制驱动器A 和 B”，最后点击确定就OK了。
这样在完成上述操作后，A和B盘在电脑中便不会显示，如果要隐藏其他的只需选择其他选项即可，如果要恢复显示，只要取消隐藏或者禁用。
